Core Features (POC)

1. Basic Application Window

  • Always-on-top rectangular window
  • Configurable size (targeting ~15% of screen width/height)
  • Basic window controls (minimize, close)
  • Semi-transparent background with UI styling

2. Virtual Assistant - Circular Oscilloscope

Assistant Preview: Circular oscilloscope with breathing gradient core

  • Outer Ring: Circular oscilloscope responding to ambient audio (radial waveform)
  • Inner Core: Animated gradient "breathing" effect
  • Audio Sync: Real-time response to ambient sound mixing
  • Visual Appeal: Hypnotic, clean aesthetic that indicates sound activity

3. Ambient Noise System

  • Multiple CC0 audio loops (rain, coffee shop, nature sounds)
  • Individual volume sliders for each sound
  • Real-time audio mixing
  • Web Audio API integration for assistant visualization

4. Timer Functionality

  • Basic countdown timer
  • Start/pause/stop controls
  • Visual feedback on assistant during timer sessions
  • Audio notification when complete

5. UI Structure with Provided Styling

  • Clean, minimal interface using provided color palette
  • Quicksand font family throughout
  • Consistent rounded corners and thick borders
  • Animated background with blurred gradient circles

Success Criteria

  • Application runs stable for 4+ hours without memory issues
  • Window stays properly always-on-top on both platforms
  • Circular oscilloscope animations are smooth and responsive
  • Ambient noise mixing works without audio artifacts
  • Timer functionality works reliably
  • UI feels polished using provided design system
  • Performance impact is negligible during normal use
